# 找不到数据类型怎么办

如果当前游戏或模型在 SSMT 里还找不到对应的数据类型，你需要把这次提取时的关键信息发给我，我才能补充支持。

## 需要提供的内容

请整理下面 3 项信息：

1. 此次提取使用的 `FrameAnalysis` 文件夹，并打包为 `.7z` 格式
2. 此次提取使用的 `DrawIB` 列表
3. 此次提取对应的是哪个游戏

## 发送方式

- `FrameAnalysis` 压缩包：通过 `QQ闪传` 发送给我
- `DrawIB` 列表和游戏名称：直接通过 QQ 发给我

::: tip 多份文件请标明对应关系
如果你这次整理了多个 `FrameAnalysis` 文件夹，以及多份对应的 `DrawIB` 列表，请务必注明谁对应谁。

否则我可能无法判断哪份数据对应哪个模型。
:::

## 提取和打包时的注意事项

::: warning 这些问题会直接影响我能不能补数据类型
- Dump 前尽量减少绿字 `IB` 的总数量，这样可以明显减小压缩包体积。
- 如果开启了 `Symlink`，部分压缩软件可能出现“越压越大”的情况，所以尽量使用 `.7z` 格式压缩。
- 如果压缩速度太慢，可以先把 Dump 文件夹移动到接在 `NVME.2` 接口上的固态硬盘里，再进行压缩。
- Dump 的必须是游戏原本的模型，不能是已经打了 Mod 的模型。
- 绝对不要使用 `指定IB列表Dump`，否则很容易遗漏关键信息，导致我无法正常补全数据类型。
- 绝对不要为了减小体积，擅自删除 `FrameAnalysis` 文件夹中的内容，否则可能丢失关键数据，导致无法提取。
:::

## 文件发送建议

::: danger 不要发网盘链接
请尽量使用 `QQ闪传` 发送。

如果文件大小超过 QQ 限制，可以先进行分卷压缩后再发送。不要给我网盘链接，否则下载可能会非常慢，甚至要拖好几天。
:::

## 后续会怎样

当我完成数据类型补充后，会推送新的 SSMT 版本并通知你。

到时候你只需要更新 SSMT，就可以正常使用了。
