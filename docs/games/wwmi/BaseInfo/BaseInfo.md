# WWMI

鸣潮 Mod 主流工具为 **SpectrumQT** 开发的 **WWMI** 系列工具：

- **特点**：稳定，适合商业化工作流。

**SSMT4** 也完整支持鸣潮 Mod 制作：

- **特点**：可使用 SSMT 全流程轮椅工作流，萌新友好。

> **注意**：SSMT 对鸣潮 Mod 的支持仅为**次要支持**，而非像米家游戏那样提供**主要支持**。部分新特性会在 WWMI-Tools 更新一段时间后才同步跟进，而非第一时间添加——不过这通常取决于你何时反馈给我（因为我不玩鸣潮）。

## WWMI 系列工具

**负责人**：[SpectrumQT](https://github.com/SpectrumQT)

### WWMI-Tools

模型提取与 Mod 生成。

- [GitHub: SpectrumQT/WWMI-Tools](https://github.com/SpectrumQT/WWMI-Tools)

### XXMI-Launcher

作为 3DMigoto 启动器使用。

- [GitHub: SpectrumQT/XXMI-Launcher](https://github.com/SpectrumQT/XXMI-Launcher)

## SSMT4 与 TheHerta4 支持

与 SSMT 支持的其它游戏一致，SSMT 全流程操作均可使用，提供轮椅级工作流体验。

## 来自 NicoMico 的建议

对于鸣潮的 Mod 制作：

- 作为一名优秀的 Mod 制作者，推荐**同时掌握 SSMT 与 WWMI-Tools** 两套流程。
- **WWMI-Tools**：更新较稳定，可用作兜底方案；但部分操作繁琐、功能简陋，属于较早时期的工具。
- **SSMT**：更新较频繁，每隔一段时间可能需要学习新用法；但熟练掌握后，制作 Mod 的时间成本非常低。

此外：

- 如果使用 Mod **大额盈利**，推荐使用 **WWMI-Tools**。
- 如果仅制作健康、阳光、正能量的 Mod 用于娱乐，且需要舒适的轮椅工作流体验，推荐使用 **SSMT4 + TheHerta4**。
- SSMT中没有内置对于鸣潮引擎修改LOD的ini文件，所以实战中一般使用XXMI-Launcher启动器 + SSMT4提取模型 + TheHerta4生成Mod流程（我懒得加那个引擎ini的替换，也懒得维护那个，直接在SSMT里把3Dmigoto目录选到XXMI-Launcher的WWMI目录下就行了）
