# SSMT4 配套文档

访问地址: https://starbobis.github.io/SSMT4-Documents/

# 开始写文档

第一次克隆后，需要进入项目目录中初始化vitepress环境
```
bun install
```

后续即可随时本地测试
```
bun run docs:dev
```
